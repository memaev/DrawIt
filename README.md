### DrawIt
Welcome to DrawIt repository. DrawIt is an officially open-source android application for shared whiteboards where you can create whiteboards, add new participants and draw together!

## Technical information:
- Minimum Android SDK API: 26
- Source code programming language: Java
- Used technologies/frameworks/libraries: Firebase (Auth|Realtime Database|Storage|Analytics), Glide, Dagger Hilt (for Dependency Injection)
